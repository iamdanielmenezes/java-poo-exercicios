# Exercício - Conta Bancária

## Descrição

Programa que realiza o cadastro de uma conta bancária informando número da conta e nome do titular.
Permite realizar depósito e saque, considerando um possível depósito inicial.
A operação de saque possui uma taxa fixa de desconto, e os dados da conta são exibidos atualizados após cada operação.

## Conceitos praticados

* Classes e objetos
* Encapsulamento
* Construtores com sobrecarga
* Métodos de depósito e saque
* Uso de Scanner
* Estrutura condicional (if)
* Método toString
* Organização em pacotes

## Objetivo

Praticar a criação de classes com atributos privados, utilização de construtores com sobrecarga e implementação de operações bancárias com regras específicas.
