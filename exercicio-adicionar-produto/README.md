# Exercício - Adicionar Produto

## Descrição

Programa que realiza o cadastro de um produto informando nome, preço e quantidade em estoque.
Permite adicionar e remover produtos do estoque e exibe os dados atualizados do produto, incluindo o valor total em estoque.

## Conceitos praticados

* Classes e objetos
* Encapsulamento
* Construtores com sobrecarga
* Métodos
* Getters e Setters
* Método toString
* Uso de Scanner
* Organização em pacotes

## Objetivo

Praticar a criação de classes com atributos privados, utilização de construtores, encapsulamento e manipulação de dados através de métodos.
