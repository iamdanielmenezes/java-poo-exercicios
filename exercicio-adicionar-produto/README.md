📄 Descrição

Programa que realiza o cadastro de um produto com nome, preço e quantidade em estoque.
Permite adicionar e remover produtos do estoque, atualizando os dados do produto e exibindo o valor total em estoque.

🧠 Conceitos praticados
Classes e objetos
Encapsulamento
Construtores com sobrecarga
Métodos de alteração de estado (add e remove)
Getters e Setters
Método toString
Cálculo com atributos da classe
Uso de Scanner para entrada de dados
Organização em pacotes
🎯 Objetivo

Praticar a criação de classes com atributos privados, uso de construtores, encapsulamento e manipulação de dados através de métodos.
