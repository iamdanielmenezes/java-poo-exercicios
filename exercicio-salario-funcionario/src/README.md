## Descrição
Programa que recebe os dados de um funcionário e calcula o salário atualizado com base em um aumento percentual informado.

## Conceitos praticados
- Criação de classes e objetos
- Manipulação de atributos
- Métodos para cálculo
- Separação entre entidade e classe principal

## Objetivo
Praticar a modelagem de uma classe representando um funcionário e aplicar lógica para cálculo de aumento salarial.
