## Descrição
Programa que recebe a cotação do dólar e a quantidade de dólares a serem comprados, realizando a conversão para reais com acréscimo de 6% de IOF.

## Conceitos praticados
- Métodos estáticos
- Separação entre entidade e classe principal
- Organização em pacotes
- Uso de Scanner para entrada de dados
- Formatação de saída com printf

## Objetivo
Praticar a criação de classes utilitárias com métodos estáticos, entrada de dados pelo usuário e organização básica de um projeto Java.
